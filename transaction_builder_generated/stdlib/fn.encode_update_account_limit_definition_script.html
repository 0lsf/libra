Optionally update thresholds of max balance, inflow, outflow for any limits-bound accounts with their limits defined at limit_address. Limits are defined in terms of base (on-chain) currency units for CoinType. If a new threshold is 0, that particular config does not get updated. sliding_nonce is a unique nonce for operation, see SlidingNonce.move for details.
